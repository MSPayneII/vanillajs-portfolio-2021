Project Background

I participated in Perficient's Bright Paths Program, which is a coding bootcamp run in collaboration with DevMountain and Hackbright Academy. The camp ran from October 2021 - February 4th, 2022, and was split into two halves. The first eight weeks focused on coding foundations, and the latter half focused on full-stack React development specialization. My 8-Ball Fortune project application served as my Coding Foundations Capstone.

Main Features

The 8-Ball Fortune Teller application has four main features:

  • Users can enter questions to ask the eight ball
  • Users will receive a randomized answer to their question via an animation displayed on the eight ball
  • Q&As will save and display on screen for users
  • Saved Q&As are erased by refreshing the browser
